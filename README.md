#  AURA

## Adaptive Universal Reality Assistant

> **Navigate Life. Not Limitations.**

AURA is an AI-powered accessibility companion that empowers people with disabilities through **Multimodal AI** and **UiPath Intelligent Automation**. Instead of simply answering questions, AURA understands the user's environment, simplifies complex government services, automates repetitive tasks, and enables greater independence through one unified platform.

---

#  The Problem

Millions of people with disabilities face unnecessary barriers every day.

* Reading and understanding government documents
* Filling lengthy application forms
* Accessing disability benefits and public services
* Navigating unfamiliar environments
* Managing important documents
* Completing repetitive digital tasks independently

Existing solutions address individual problems, requiring users to switch between multiple applications.

**There isn't one intelligent companion that understands, guides, and acts.**

---

#  Our Solution

AURA combines **Multimodal AI** with **UiPath Intelligent Automation** to create a single accessibility platform that helps users navigate, understand, and automate everyday tasks.

Rather than acting as another chatbot, AURA functions as an intelligent companion that supports users throughout their daily lives.

---

#  Key Features

##  Reality Companion

* Real-time scene understanding
* Environmental awareness
* Voice-guided assistance
* Object and obstacle recognition

##  Government Assistant

* Scan physical government forms
* Upload official documents
* Explain paperwork in simple language
* Recommend eligible government schemes
* Guide users through applications
* Track application status

##  Intelligent Document Understanding

Supports documents such as:

* Aadhaar Card
* Disability Certificate
* Medical Reports
* PAN Card
* Government Letters
* Passport Photos

Aura automatically identifies the document, extracts important information, and explains difficult content in simple language.

##  Aura Profile Vault

Stores verified user information securely so users never have to repeatedly upload the same documents or re-enter the same information.

##  Eligibility Engine

Based on the user's profile, Aura recommends government schemes and services including:

* Disability Certificate
* Scholarships
* Travel Concessions
* Assistive Device Schemes
* Welfare Programs

##  Intelligent Automation

Using UiPath, Aura automates repetitive workflows such as:

* Portal navigation
* Form filling
* Document uploads
* Application submission
* Receipt generation
* Status tracking

## Emergency Assistance

Provides quick, accessible emergency support designed for users with different accessibility needs.

#  How AURA Works

### 1️⃣ Understand

Users interact through voice, text, camera, or document upload.

↓

### 2️⃣ Analyze

Gemini understands user intent, analyzes images and documents, and determines the next steps dynamically.

↓

### 3️⃣ Organize

Extracted information is securely stored inside the Aura Profile Vault for future use.

↓

### 4️⃣ Recommend

Aura identifies relevant government schemes and services based on the user's profile.

↓

### 5️⃣ Automate

UiPath automates repetitive workflows including application filling, document uploads, and status tracking.

#  Installation

```bash
git clone https://github.com/nisithaa-03/AURA-.git
cd AURA-
npm install
```

Create a `.env.local` file:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application:

```bash
npm run dev
```
